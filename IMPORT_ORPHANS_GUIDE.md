# استيراد ملف الأيتام

## ما تم إضافته

- زر واضح في لوحة الإدارة باسم: **اختيار ملف Excel واستيراده**.
- زر لتحميل قالب Excel منظف: **تحميل القالب المنظف**.
- قراءة تلقائية للملف حتى لو كان صف العناوين ليس أول صف، بشرط وجود عمود: `اسم الطفل رباعي`.
- تجاهل الصفوف الفارغة تلقائيًا.
- تجاهل الصفوف التي لا تحتوي اسم طفل.
- منع التكرار داخل نفس الملف إذا تكرر نفس اسم الطفل مع نفس رقم الوصي.
- عرض ملخص بعد الاستيراد: عدد المستورد، عدد الصفوف الفارغة، وعدد الصفوف المتجاهلة.

## مكان القالب

القالب موجود داخل المشروع في:

```txt
public/templates/orphans_cleaned_template.xlsx
```

ويظهر في لوحة الإدارة من زر:

```txt
تحميل القالب المنظف
```

## ملاحظات مهمة

1. لا تغيّر أسماء الأعمدة الأساسية.
2. اكتب أرقام الجوال كنص حتى لا يختفي الصفر الأول.
3. استخدم تاريخ واضح مثل `2015-04-18`.
4. القيم المقبولة:
   - الجنس: `ذكر`، `أنثى`، `غير محدد`
   - حالة اليتيم: `يتيم الأب`، `يتيم الأم`، `يتيم الأبوين`، `غير محدد`
   - حالة الكفالة: `بانتظار كافل`، `مكفول`، `متوقف`
   - حالة الملف: `جديد`، `جديد بانتظار المراجعة`، `قيد المراجعة`، `مقبول`، `مرفوض`، `مكتمل`
   - العملة: `شيكل`، `دولار`، `دينار`، `غير محدد`

# البيانات الناقصة المطلوبة منك

أغلب الملفات جاهزة. المتبقي فقط بيانات لا أستطيع معرفتها من خارج حسابك:

## 1) حساب المدير في Supabase Auth

بعد إنشاء مستخدم المدير من لوحة Supabase، أحتاج منك:

- User ID الخاص بالمستخدم المدير
- بريد المدير المعتمد
- الاسم الذي تريد ظهوره في النظام

مثال:

```txt
Admin User ID: 00000000-0000-0000-0000-000000000000
Admin Email: example@example.com
Admin Name: Super Admin
```

## 2) اسم المنصة النهائي

القيمة الحالية داخل الواجهة:

```txt
منصة كفالة الأيتام
```

إن أردت اسمًا آخر، أرسله لي.

## 3) العملة الافتراضية

القيمة الحالية:

```txt
شيكل
```

الخيارات الموجودة: شيكل، دولار، دينار، غير محدد.

## 4) قائمة المحافظات / المدن

القائمة الحالية عامة. إذا تريد قائمة محددة خاصة بمؤسستك أرسلها لي.

## 5) سياسة الملفات

الوضع الحالي:

- حجم الملف الأقصى: 10MB
- الأنواع المسموحة: PNG, JPG, WEBP, PDF
- Bucket خاص وليس عامًا

إذا تريد تغيير الحجم أو الأنواع، أرسل المطلوب.
